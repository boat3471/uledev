# uledev
用于构建邮乐个人开发环境

####使用说明
1. 安装Git (自行百度)
2. 安装Nodejs (自行百度)
3. 创建个人开发目录, 一般使用个人姓名全拼命名
4. 在开发目录打开cmd, 或打开cmd后定位到开发目录
5. 安装模块 uledev
	安装命令: npm install -g uledev
	uledev init 命令说明:
	uledev init 命令可以初始化个人开发环境, 创建.conf文件夹, 静态资源目录和页面资源目录

6. 执行命令行: uledev init
7. 使用nodejs命令, 启动uledev服务器:
	启动.conf目录下 uledev 文件即可
8. 啊是打算
